## Featured Project

### PFE: Inventory Management System (Kromberg & Schubert)
A comprehensive web application for monitoring stock levels, processing transactions as well as keep track of their status. It reduces manual oversight and enhances decision-making with data-driven insights.

**Key Features:**
- Real-time dashboard displaying transaction trends, supplier analytics, and stock visualizations.
- Role-based access control (e.g., admin for reports, users for transactions) with secure Angular interfaces.

**Tech Stack:** Java/Spring Boot (backend APIs and logic), Angular/TypeScript (responsive frontend), SQL database (for products/transactions), Mermaid for system diagrams.
